05-数组 排序 查找

对应内容：P156 - P190

数组

  • 概念
    • 数组可以存放多个同一类型的数据，数组也是一种数据类型，是引用类型。
  • 使用方式
    • 使用方式1 动态初始化
      • 定义数组
        • 语法:数据类型 数组名[] = new 数据类型[大小]
        • int a[] = new int[5];
        • (int [] a = new int[5];)两者一样的
      • 引用数组
        • 数组名[下标/索引] （下标从0开始）
    • 使用方式2 动态初始化
      • 先声明数组
        • 语法:数据类型 数组名;
        • int a[]; int[] a;
      • 创造数组
        • 语法:数组名 = new 数据类型[大小];
        • a = new int[10];
    • 使用方式3 静态初始化
      • 初始化数组
        • 语法: 数据类型 数组名[] = {元素值，元素值};
  • 数组注意事项
    1. 数组是多个同类型数据的组合，以实现对这些数据的统一管理
      • e.g.：int a[] = {1,2,3,4,1}; error double->int
      • e.g.:double b[] = {1.1, 2,3}; true int->double
    2. 数组中的元素可以是任何数据类型，包括基本类型和引用类型
    3. 数组创建后，如果没有赋值，有默认值
      1. int 0, short 0, byte 0, long 0
      2. float 0.0, double 0.0, char \u0000
      3. boolean false, String null
    4. 使用数组的步骤：
      1. 声明数组并开辟空间
      2. 给数组的每个元素赋值
      3. 使用数组
    5. 数组的下标从0开始
    6. 下标必须在指定范围内使用，否则报：下标越界异常
    7. 数组属于引用类型，数组型数据是对象(object)

  • 数组赋值机制
    - 数组整体赋值在默认情况下是引用传递，赋的值是地址，赋值方式为引用传递

  • 数组拷贝
    • int[] arr1 ={1,2,3};
    • int[] arr2 = new int[arr1.length];
    • for(int i = 0; i < arr1.length; i++)
      • { arr2[i] = arr1[i];}
  • 数组反转
    • 前后依次翻转
    • 逆序赋值（创建新数组）

  • 数组扩容
    - int[] arr = {1,2,3};
    - int[] arrNew = new int[arr.length+1];
    - for(){...}
    - arrNew[arrNew.length-1] = 4;
    - arr = arrNew;
    - //java中原数组似乎不用自己释放，Java有垃圾回收机制

排序

  • 分类
    • 内部排序
      • 将所有需要处理的数据都加载到内存存储器中进行排序
      • 冒泡，插入，选择
    • 外部排序
      • 数据量过大，无法全部加载到内存中，需要借助外部存储进行排序
      • 合并排序，直接合并排序法
  • 冒泡排序

查找

  • 常用的查找
    • 顺序查找
    • 二分查找

二维数组

  • int[][] arr = {{1,1,1}{1,1,1}{1,1,1}};
  • 使用方式
    • 使用方式1 动态初始化
      • 定义数组
        • 语法:数据类型 数组名[] = new 数据类型[大小]
        • int a[][] = new int[5][6] ;
        • (int [] [] a = new int[5][6] ;)两者一样的
      • 引用数组
        • 数组名[下标/索引] （下标从0开始）
    • 使用方式2 动态初始化
      • 先声明数组
        • 语法:数据类型 数组名;
        • int a[][] ; int[] [] a;
      • 创造数组
        • 语法:数组名 = new 数据类型[大小][大小] ;
        • a = new int[10];
    • 使用方式3 动态初始化—-列数不确定
      • 这部分不太记得了
    • 使用方式3 静态初始化
      • 初始化数组
        • 语法: 数据类型 数组名\[\]\[\] = \{\{\},\{\}\};
  • 使用细节
    1. 二维数组的声明方式：
      • int[][]y; int []y[]; int y[][];
    2. 二维数组实际上是由多个一维数组组成的，各个一维数组的长度可以相同，也可以不同
      1. int map\[\]\[\] = \{\{1,2\},\{3,4,5\}\};
